# Popmundo Scripts

This is my set of scripts for [Popmundo roleplaying game](http://popmundo.com).

## Requiments

### Firefox Web Browser

If you use Firefox, you must have [GreaseMonkey](https://addons.mozilla.org/en-us/firefox/addon/greasemonkey/) extension installed.

### Chrome Web Browser

If you use Chrome, you must have [TamperMonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) extension installed.

## Scripts

### Joe's SpeedCalling for Popmundo

Forked from SpeedCallig for Popmundo, this is an updated script that allows you to call all the people in your Phonebook with a single click.
